Cyber Wave is a security related cyber project aiming to provide a gamification of 
traditional cyber security training, it will provide the users with scenarios which work 
to earn them money, once the player has answered all of the questions, they are 
able to buy different moves from the shop which will make playing the game easier. 
The fundamental gameplay works by having a server in the centre of the map and 
different types of enemies are coming to attack that server in waves. The player’s 
goal is to protect the central server. 

In order to play download the build file from the git hub, the database and the server code.
start the server and database running and its ready to go :)
